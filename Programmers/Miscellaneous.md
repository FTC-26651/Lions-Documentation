## Motor Stalling
Motor Stalling is actually useful. If you have a physical block so the motor can't go beyond it, and want to see when the motor is up against it, you can see when it stalls. A stalled motor will take more current than usual. So, if you check the current of the motor and it's over whatever current limit you set it to, you can be quite confident that it is stalling. And, surprisingly, stalling the motors doesn't actually damage them.
